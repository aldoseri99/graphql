## 📋 Overview

This project queries the [Reboot01](https://learn.reboot01.com) GraphQL API to display personalized progress data through a cold, corporate, retro-terminal UI — inspired by the show *Severance*.

---

## ✨ Features

### 🔐 Authentication
- Login with **username** or **email** + password
- JWT-based session via Bearer authentication
- Credentials encoded with Base64 for the signin endpoint
- Error feedback for invalid credentials
- SVG graph


### 📊 Statistics (SVG Graphs)
1. **XP Over Time** — Line graph tracking cumulative XP earned across your journey
2. **Audit Ratio Chart** — Visual breakdown of audits done vs. audits received
3. **Projects Pass/Fail Pie Chart** — Proportional view of passed vs. failed projects
4. **Skills Radar Chart** — Multi-axis radar showing your skill distribution
